Polimerização por reação de adição

O politetrafluoroeteno forma-se a partir do tetrafluoroeteno. A reação de polimerização do tetrafluoroeteno ocorre com transformações das ligações duplas C=C em ligações simples C-C e aumento do número de átomos a que cada carbono está ligado:

De 3, no monómero, passa para 4, no polímero. Por isso, este tipo de polimerização chama-se polimerização de adição

Como as ligações C-F são muito estáveis, este polímero é muito pouco reativo, o que o torna útil em medicina:

Válvulas cardíacas, pacemakers e próteses

Por outro lado, o facto das ligaçoes C-F serem fortemente polares determina ligações intermoleculares fortes e pontos de fusão elevados

Iniciação, propagação e terminação

Nas polimerizações por reação de adição, o primeiro passo utiliza um iniciador que origina dois radicais

R2 -> 2R·

Usando como exemplo a síntese do polietileno, o radical R· associa-se com uma molécula de eteno, originando um novo radical

R· + CH2=CH2 -> R-CH2-CH2·

Este novo radical acaba por reagir com outra molécula de eteno

R-CH2-CH2· + CH2=CH2 -> R-CH2-CH2-CH2-CH2·

Este processo repete-se inúmeras vezes na chamada propagação. Eventualmente, este termina quando dois radicais longos se juntam, originando uma macromolécula estável

R-CH2-CH2-CH2-CH2· + R-CH2-CH2-CH2-CH2·

↓

R-CH2-CH2-CH2-CH2-CH2-CH2-CH2-CH2-R
